What is HTML?

    Tim Berners-Lee invented HTML in 1991.

  • HTML stands for HyperText Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content

<!DOCTYPE html>
It represents the document type, and helps browsers to display web pages correctly. It defines that this document is an HTML5 document
<html>
It's the root element of an HTML page
<head>
It contains meta information abot the HTML page
<meta>

Purpose of Meta Tags

  • Provide information about the webpage, like character encoding or page description.
  • Ensure your site is SEO-friendly.
  • Improve the user experience on mobile devices.
  • Provide better sharing previews on social media.
<title>
It specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
<body>
It defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
