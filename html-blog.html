Empty HTML Elements
HTML elements with no content are called empty elements. <br> is an empty element.
href attribute
The href attribute specifies the URL of the page the link goes to
src attribute
it specifies the path of the image to be displayed
width, height attribute
provide size information for images
alt attribute
provides an alternate text for an image
style attribute
It's used to add styles to an element, such as color, font, size and more.
lang attribute
It declares the language of the web page
title attribute
It defines some extra information about an element. It's used for showing tooltip in web psge
<pre>
It defines preformatted text. It's displayed in a fixed-width font, and preserves both spaces and line breaks: 
                My Bonnie lies over the ocean.

                My Bonnie lies over the sea.

                My Bonnie lies over the ocean.

                Oh, bring back my Bonnie to me.
